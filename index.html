MENU RESTAURANTE 1

Se realizó una página web para el menú del restaurante "La Cierra". Esta página fue diseñada con un enfoque en la presentación visual de los platillos, utilizando HTML y CSS para crear un diseño atractivo y responsive. La estructura permite que los usuarios naveguen fácilmente por las diferentes secciones del menú, mostrando los detalles de cada platillo de manera clara y accesible en cualquier dispositivo.

Pagina Crunchyrol

Se creó una página web inspirada en el estilo de Crunchyroll, diseñada para ofrecer una experiencia de usuario intuitiva y atractiva. La página fue construida utilizando HTML y CSS, enfocándose en la organización y presentación de contenido multimedia, como anime y videos. La interfaz es responsive, asegurando que los usuarios puedan acceder al contenido fácilmente desde cualquier dispositivo, con una navegación fluida y un diseño visual que refleja la estética de Crunchyroll.
